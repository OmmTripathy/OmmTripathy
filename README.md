### 🤖 RAG Chatbot — AI Document Intelligence
> Upload any PDF, ask questions — retrieves context via vector search before generating answers with an LLM.

- Built end-to-end RAG pipeline: PDF ingestion → chunking → HuggingFace embeddings → ChromaDB retrieval → LLM generation
- Achieved grounded, hallucination-resistant responses using context-aware prompting with Mistral AI
- Deployed as an interactive Streamlit app with clean UX for non-technical users
